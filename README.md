# Metode Eksperimen untuk Manajemen Proyek Konstruksi

Mata kuliah Master yang mengajarkan metodologi penelitian eksperimental, pengumpulan data, analisis statistik, dan pengambilan keputusan berbasis bukti dalam konteks manajemen proyek konstruksi.

## Tujuan

- Memahami desain eksperimen dan metodologi penelitian
- Menerapkan metode ilmiah untuk manajemen proyek konstruksi
- Menguasai analisis data dan interpretasi hasil
- Merancang studi empiris yang valid dan reliable

## Modul Pembelajaran

1. **Pengantar Metode Eksperimen** - Fondasi penelitian dan desain eksperimental
2. **Desain Eksperimen** - Desain riset, hipotesis, kontrol variabel, randomisasi
3. **Manajemen Data Proyek** - Pengumpulan, validasi, dan etika data
4. **Exploratory Data Analysis** - Deskripsi, deteksi outlier, visualisasi
5. **Pemodelan dan Simulasi** - Model matematika, regresi, simulasi
6. **Visualisasi dan Keputusan** - Teknik visualisasi dan pengambilan keputusan berbasis data
7. **Studi Kasus Proyek** - Analisis kasus nyata dan praktik terbaik
